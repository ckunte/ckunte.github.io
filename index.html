Another example is about conditional formatting. I like highlighting terms like tbc or tbd, so I can return to them later. This code below highlights and italicises any of these words: tbc, tbd. I can add it to a template without modifying the report’s main content; neat!

show regex("tb[c,d]"): set text(fill:red, style:"italic")

24W26: Making room

Out of options to get Typst on my die-hard MacBook Air (2011) via either Homebrew or MacPorts, I found this fantastic package called typst-py. To simplify further, I wrote an alias to complement my ctd.py script:

alias tc='_tc() { python3 /path-to/ctd.py -f $1 ;}; _tc'

With this, I typeset like so, and save a few key strokes:

$ tc note.typ
Compiling note.typ to note.pdf...done.
