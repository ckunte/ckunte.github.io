zellij terminal workspace

18 Dec 2025, 1 min

A month or so ago, I began using zellij — as my workflow at work got complicated, I needed a single window but with multiple terminal prompts on the work laptop running debian (via WSL). I set a layout, which I was previously setting-up manually at every terminal session — a pane split horizontally, like so, and saved it to ~/.config/zellij/layouts/ck_layout.kdl:

layout {
    pane split_direction="horizontal" {
        pane // top pane
        pane // bott pane
    }
}

and called this in within ~/.config/zellij/config.kdl like so:

default_layout "ck_layout"

Pretty simple. Now every terminal session loads with this pre-set layout, and it saves me a few keystrokes. I like this simple split so much that I’ve ported this layout to my daily driver as well.
