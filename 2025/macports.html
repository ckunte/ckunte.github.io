MacPorts

25 Feb 2025, 2 min

MacPorts keeps my 13 year old MacBook Air 11 alive and usable. While no longer my primary computer, I find the laptop on the go useful. So when Typst v0.13.0 got released last Wednesday, I looked up to see if the port was updated. It wasn’t. That was last Friday. Not wanting to wait for someone else to get the wheels turning, I decided to send a pull request myself. The maintainers merged it on Sunday, after incorporating a couple of helpful suggestions.

Update (Oct 24): Being a single binary, Typst offers a command line update option. One just needs to run this to be updated to the latest. How cool is that?1

sudo typst update

In the above, sudo is needed if Typst is saved in a location like /usr/local/bin for write access.

  1. However, the MacPorts version does not support the update option. So, I uninstalled the MacPorts version of Typst, and downloaded the typst binary. ↩
