Build python

Installing python on Windows has gotten better. Installing and updating modules is a breeze too.

I recently had to reinstall a python instance, had to set up my text editor for it in the process. Sublime Text prefers a path to python binary defined in a build file. I named this one Python-win.sublime-build and placed it under User packages.

{   
  "shell_cmd" : "python \"$file_name\"",
  "selector" : "source.py",
  "path" : "C:\\path_to_python_binary_folder;$path",
  "working_dir" : "$file_path"
}

Now a python script can be run directly from within the editor by selecting Python-win from Build options, and then hitting Ctrl+B.
